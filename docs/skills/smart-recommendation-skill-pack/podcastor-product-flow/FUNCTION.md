# 功能说明

`podcastor-product-flow` 是面向 Podcastor / AI 播客工作站的专用产品体验流。它从创作者需求、首页入口、核心编辑器、资产复用、分发增长到验收指标逐步拆解。

## 核心能力

- 识别目标创作者、核心痛点、工作站定位、一期范围和成功标准。
- 设计首页创作入口：Generate Script、Upload Script、Upload Audio。
- 拆解核心编辑器：脚本预览、主持人选择、音色选择、音频生成、视频生成和编辑动作。
- 规划资产复用与分发：脚本、音频、视频、封面、历史项目和导出。
- 建立增长路径、验收指标和交付边界。

## 输出产物

- 播客需求诊断。
- 首页入口方案。
- 核心编辑器流程。
- 资产复用与分发方案。
- 增长和验收指标。
- 产品蓝图和工程交付拆解。
