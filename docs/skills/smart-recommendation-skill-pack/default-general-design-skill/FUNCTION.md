# 功能说明

`default-general-design-skill` 是默认通用 Skill，用于用户需求还不清晰、项目资料较多或不知道应该调用哪个专用 Skill 的场景。它先诊断需求，再检索项目知识库，最后推荐下一步 Skill 或整理目标 Skill 输入。

## 核心能力

- 判断需求清晰度和任务类型。
- 检索项目知识库、历史需求、业务流程、竞品资料和设计规范。
- 识别缺失信息和风险假设。
- 提出最多 3 个高价值澄清问题。
- 推荐 Skill 链或整理目标 Skill 的输入。
- 标注知识库事实和 AI 推断。

## 输出产物

- 当前理解。
- 需求类型。
- 已知信息。
- 缺失信息。
- 推荐 Skill。
- 下一步动作。
