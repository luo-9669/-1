## 一、核心设计原则

### 1.1 效率优先

电商批量操作路径必须尽可能短，核心 AI 生成按钮需要具备最高视觉权重。高频操作按钮前置，减少多层弹窗跳转，批量功能独立区块展示，适配商家多商品批量出图场景。

### 1.2 轻量化科技

采用低饱和渐变、2px 统一细描边线性图标和柔和弱阴影。避免厚重渐变、立体浮雕、高饱和刺眼色块。AI 功能仅使用浅紫色作为点缀，大面积页面维持蓝、白、浅灰的干净基调。

### 1.3 层级清晰

页面强制区分「导航 - 工具 - 画布 - 结果面板」四大视觉分区。通过背景色、分割间距和阴影强度建立层级，让用户快速区分操作区与预览区，降低工具学习成本。

### 1.4 品牌统一

全平台复用品牌蓝作为交互主色。AI 智能相关功能统一使用浅紫科技辅助色。全站点色彩、字体、圆角和图标尺寸保持一致，避免出现差异化自定义样式。

### 1.5 无障碍可读

正文文字与背景对比度不低于 4.5:1。所有按钮、输入框、单选、复选和下拉选项的最小点击热区为 44×44px。禁用浅灰小字叠加浅背景；色盲模式下，状态色需要配合形状或图标辅助识别。

### 1.6 模板管控规范

企业团队版支持品牌模板锁定能力，可固定品牌色组、默认字体、LOGO 图层和固定尺寸。普通运营用户仅允许修改产品图与营销文案，禁止修改品牌核心视觉元素。

## 二、色彩系统规范

### 2.1 品牌主色

品牌主色是全局核心交互色，用于主按钮、选中态、导航高亮和关键路径操作。

| 类型 | HEX | RGB | 使用场景 |
| --- | --- | --- | --- |
| 品牌蓝 Primary | `#1677FF` | 22, 119, 255 | 主按钮、导航高亮、Tab 选中态、品牌 LOGO、AI 生成核心操作、导出 / 保存按钮、链接文字 |
| Primary Hover | `#0958D9` | 9, 88, 217 | 按钮悬浮、可点击文字高亮、卡片 hover 边框高亮 |
| Primary Active | `#0040C9` | 0, 64, 201 | 按钮点击按压态、当前激活标签、选中模板边框 |

### 2.2 AI 科技辅助色

AI 辅助色仅用于智能 AI 相关模块，不做大面积背景填充。

| 类型 | HEX | 使用场景 |
| --- | --- | --- |
| AI 浅紫 Accent | `#7B61FF` | AI 模块标签、智能工作流、Agent 面板、技能中心、提示词输入框聚焦边框、AI 生成标签 |
| Accent Hover | `#6346E5` | AI 功能卡片悬浮背景、AI 操作按钮 hover |

### 2.3 功能状态色

| 状态 | HEX | 使用场景 |
| --- | --- | --- |
| Success 成功 | `#00B42A` | AI 生成完成、导出成功、素材上传通过、参数校验正确、保存成功提示 |
| Warning 警告 | `#FF7D00` | 素材版权提醒、图片尺寸不兼容、会员额度不足、图片分辨率过低、模板缺失素材 |
| Error 错误 | `#F53F3F` | AI 生成失败、图片上传异常、参数填写错误、网络请求失败、文件格式不支持 |
| Info 提示 | `#86909C` | 功能说明提示、空状态引导文案、底部小字备注、非重要辅助说明 |

### 2.4 中性灰度系统

页面、卡片、分割线和文字统一复用以下灰度色，禁止随意新增自定义灰色。

| 色阶 | HEX | 用途 |
| --- | --- | --- |
| N0 纯白 | `#FFFFFF` | 页面主背景、画布默认底色、弹窗底色、下拉面板底色 |
| N1 | `#F7F8FA` | 次级卡片背景、侧边工具面板、未选中画布模板缩略图背景、列表隔行底色 |
| N2 | `#E5E6EB` | 模块分割线、输入框默认边框、卡片分割线、弱分隔边框、滑块轨道底色 |
| N3 | `#C9CDD4` | 禁用状态文字、次要描边边框、空白占位图标、未激活滑块圆点 |
| N4 | `#86909C` | 辅助说明文字、备注小字、次要标签文本、占位提示文字 |
| N5 | `#4E5969` | 正文次要标题、表单 Label、列表次要描述文本、二级菜单文字 |
| N6 | `#272E3B` | 全局正文主文字、侧边导航菜单常规文字、面板说明文字 |
| N7 近黑 | `#1D2129` | 页面一级大标题、弹窗头部标题、画布顶部标题、卡片主标题 |

### 2.5 电商画布输出配色约束

- 单张电商成品物料主色数量不超过 3 种，不同主色明度差值不低于 30%，禁止多高饱和撞色堆砌。
- 白底商品图中，产品主体与纯白画布背景对比度不低于 7:1，保证电商平台检索和手机端识别清晰。
- 营销海报中的强调色仅允许用于价格数字、活动按钮、限时角标标签，不可铺满大面积背景。
- 企业品牌模板自动绑定预设品牌色组，普通用户仅可修改文案与产品图片，禁止替换品牌主色和辅助色。

## 三、字体排版规范

### 3.1 全局字体栈

系统页面 UI 文字：

```css
font-family: "Source Han Sans SC", "Microsoft YaHei", Inter, system-ui, -apple-system, sans-serif;
```

画布设计素材字体使用平台内置商用授权字体库，分为标题字体、正文宋体、手写字体、粗黑体四大类。企业团队支持预设品牌专属字体锁定，模板不可随意替换。

### 3.2 字号阶梯

字号以 14px 正文为基准，采用 4px 递进。

| 层级 | 字号 | 字重 | 行高 | 应用场景 |
| --- | --- | --- | --- | --- |
| H1 页面大标题 | 28px | 600 | 38px | 页面顶部总标题、AI 创作首页大模块标题、画布项目名称 |
| H2 模块标题 | 24px | 600 | 32px | 侧边栏分组大标题、弹窗头部主标题、工具分区标题 |
| H3 卡片标题 | 20px | 500 | 28px | 模板卡片标题、功能板块标题、批量任务列表标题 |
| H4 子标题 | 18px | 500 | 24px | 表单分组小标题、画布内素材分组标题、筛选栏标题 |
| Body1 正文 | 16px | 400 | 22px | 输入框填写文字、面板详细说明、列表主体文本、参数配置文本 |
| Body2 默认正文 | 14px | 400 | 20px | 全局默认基础正文、侧边菜单文字、参数辅助说明、表格内容 |
| Caption 辅助小字 | 12px | 400 | 16px | 剩余额度、版权信息、时间戳、底部备注、图片尺寸标注 |
| Label 标签小字 | 12px | 500 | 16px | 品类标签、AI 智能标记、状态角标、筛选标签 |

### 3.3 排版约束

- 标题类文本字间距为 `-0.01em`，正文文本字间距为 `0em`，辅助小字和标签字间距为 `+0.01em`。
- 画布物料内置九宫格、三等分和黄金分割辅助线，商品主体应放置在视觉黄金交点附近，避免画面失衡。
- 电商主图导出时，产品主体四周预留不低于 80px 空白安全边距，防止平台弹窗、水印和标签遮挡商品。
- 卡片、列表内多行文本最多展示 3 行，超出文字自动折叠省略，hover 时通过气泡展示完整内容。

## 四、间距与栅格系统

### 4.1 基础单位

统一基础栅格单位为 4px。页面所有内边距、外边距、组件宽高、圆角尺寸和图标尺寸必须为 4px 的整数倍，禁止使用奇数像素值。

### 4.2 全局间距阶梯

| 标识 | 数值 | 使用场景 |
| --- | --- | --- |
| XS | 4px | 图标与文字间距、卡片内部微小间隙、单选框 / 复选框内边距、标签上下间距 |
| S | 8px | 列表单行内间距、按钮左右留白、分组内小型分隔间隙、输入框内上下边距 |
| M | 16px | 卡片统一 padding、模块内边距、弹窗内容区边距、表单分组间距 |
| L | 24px | 模块与模块横向 / 纵向分隔距离、页面左右安全边距、画布工具栏间距 |
| XL | 32px | 页面顶部 / 底部外边距、页面大区块分割间距、弹窗与页面间距 |
| XXL | 48px | 首页首屏大区块上下间距、左侧导航栏与中间工具区间隔、画布与操作面板间隔 |

### 4.3 页面布局栅格

标准页面采用三栏固定布局：

- 左侧导航菜单栏：固定宽度 240px，高度 100%，独立滚动；承载产品一级菜单、AI 功能分类、个人中心和团队管理入口。
- 中间 AI 工具操作区：自适应弹性宽度；承载品类筛选、模板选择、AI 参数配置和批量操作功能。
- 右侧画布 / 生成结果预览区：最小宽度 600px，支持拖拽分割线调整宽度；包含画布编辑器、生成结果缩略图和素材库面板。

### 4.4 画布内部网格

- 画布基础网格采用 12 等分响应式网格，适配淘宝、拼多多、京东、亚马逊等主流平台商品尺寸。
- 辅助线与标尺刻度间隔统一为 20px，颜色为 N3，线宽为 1px。
- 图层对齐吸附阈值为 8px，自动吸附画布边缘、辅助线和其他图层边界。

## 五、圆角规范

所有组件圆角遵循 4px 梯度，不允许自定义不规则圆角。

| 规格 | 数值 | 适用组件 |
| --- | --- | --- |
| Small 小圆角 | 4px | 单行输入框、状态标签、小型文字按钮、单选 / 复选卡片、滑块轨道 |
| Middle 中圆角 | 8px | 功能卡片、弹窗容器、工具侧边面板、模板缩略图、下拉菜单容器 |
| Large 大圆角 | 16px | 主画布容器、首页大功能模块、AI 生成大卡片、大面积操作面板 |
| Full 全圆角 | 999px | 用户头像、状态圆点、胶囊标签按钮、圆形图标按钮 |

## 六、通用组件规范

### 6.1 按钮组件

所有按钮统一最小点击热区为 44px 高，基础过渡动画为 `0.2s ease`。

| 类型 | 样式 | 使用规则 |
| --- | --- | --- |
| Primary 主按钮 | 背景 `#222529`；hover `#1D2129`；active `#111318`；圆角 8px；水平内边距 16px；垂直内边距 8px；最小尺寸 88×44px；文字白色 14px / 500 | 流程通主按钮，用于核心提交、保存、导出、生成等主路径操作 |
| Brand 品牌辅助按钮 | 背景 `#1677FF`；hover `#0958D9`；active `#0040C9`；尺寸、圆角、内边距同主按钮 | 用于品牌链接、选中态、焦点态、导航高亮，不作为默认主按钮 |
| Accent AI 专属按钮 | 背景 `#7B61FF`；hover `#6346E5`；尺寸、圆角、内边距同主按钮 | 仅用于 AI 生成、智能复刻、Agent 批量处理等智能功能 |

黑色主按钮是默认最高优先级按钮样式，品牌蓝只作为辅助强调色使用。
| Default 次要按钮 | N0 白底，N2 1px 边框，文字 N6；hover 时边框变为 `#1677FF`，背景为 `#E8F3FF` | 用于次级操作、取消、筛选等 |
| Text 文字按钮 | 无边框、无背景，文字 `#1677FF`；hover 时显示下划线且透明度为 0.8 | 用于轻量跳转和辅助操作 |
| Disabled 禁用态 | 背景 N1，文字 N3，取消 hover 交互，鼠标指针为 `not-allowed` | 用于不可操作状态 |

### 6.2 输入框组件

- 固定高度：40px。
- 默认边框：N2 1px；聚焦状态：`#1677FF` 2px 描边。
- 圆角：4px。
- 内边距：左右 12px，上下 8px。
- 占位文字颜色：N4；输入正文颜色：N6。
- 错误态：边框 `#F53F3F`，下方展示 12px 错误提示文案。

### 6.3 卡片容器

- 基础背景：N0 / N1，按层级区分使用。
- 圆角：8px。
- 基础阴影：`0 2px 8px rgba(0, 0, 0, 0.06)`。
- 标准内边距：16px。
- 内部分割线：N2 1px 全宽横线。

### 6.4 Tab 标签栏

- Tab 单项高度固定 44px，左右内边距 16px。
- 未选中 Tab：文字 N5，无底部线条。
- 选中 Tab：文字 `#1677FF`，底部 2px 主色下划线。
- Tab 容器底部带 1px N2 分割线。

### 6.5 图标系统

- 统一风格：线性 2px 描边，无填充、无渐变、无圆角图标。
- 标准尺寸：16px / 20px / 24px，全部为 4px 倍数。
- 常规功能图标颜色为 N5。
- AI 智能模块图标颜色为 `#7B61FF`。
- 操作按钮图标颜色为 `#1677FF`。
- 禁用图标颜色为 N3。

### 6.6 下拉菜单与筛选面板

- 背景：N0。
- 圆角：8px。
- 阴影：使用弹窗浮层阴影。
- 列表项高度：40px。
- Hover 背景：N1。
- 选中项左侧使用 4px 主色竖线标记。

### 6.7 进度条与骨架屏

- AI 生成长任务进度条：轨道背景 N1，填充条 `#1677FF`，高度 6px，圆角 999px。
- 加载骨架屏：使用 N1 到 N2 的循环渐变，不使用复杂动画，降低性能消耗。

### 6.8 Toast 轻提示

- 容器圆角：8px。
- Padding：12px 20px。
- 位置：页面顶部居中。
- 显示时长：2s 自动消失。

| 类型 | 底色 | 文字 / 图标色 |
| --- | --- | --- |
| 成功 | `#E6FFEA` | `#00B42A` |
| 警告 | `#FFF7E6` | `#FF7D00` |
| 错误 | `#FFECEC` | `#F53F3F` |

## 七、画布编辑器专项规范

- 画布基础底色默认为 `#FFFFFF`，支持透明素材棋盘格预览，棋盘格使用灰白 20px 方格。
- 标尺与辅助线颜色统一为 N3，线宽 1px；吸附开启时辅助线高亮为 `#1677FF`。
- 图层选中边框为 2px `#1677FF`，圆角 8px，图层缩放控制点使用主色填充。
- 画布悬浮工具栏固定在画布右上角，使用半透 N0 背景、8px 圆角和弱层级阴影。
- 导出尺寸预设固定清单：淘宝主图 800×800、拼多多主图 750×750、京东 800×800、亚马逊白底 1000×1000、详情页 790 宽通栏。预设列表不可随意新增。
- 图层管理面板放置在左侧图层列表中，隐藏 / 锁定图标统一使用线性风格，锁定图层透明度降低至 60%。

## 八、阴影分层规范

统一使用柔和弱阴影，避免厚重深色阴影。

| 层级类型 | 阴影参数 | 使用场景 |
| --- | --- | --- |
| 基础卡片浅层阴影 | `0 2px 8px rgba(0, 0, 0, 0.06)` | 模板卡片、侧边工具面板、列表卡片 |
| 弹窗浮层高层级阴影 | `0 8px 24px rgba(0, 0, 0, 0.12)` | 全局弹窗、下拉菜单、筛选浮层、批量操作弹窗 |
| 画布控件微型阴影 | `0 1px 4px rgba(0, 0, 0, 0.08)` | 画布内悬浮工具栏、迷你悬浮按钮、图层操作浮窗 |

## 九、全局交互动效规范

- 所有 hover 和点击切换使用 `0.2s ease`，避免过快或过慢动画。
- AI 生成、图片上传仅使用渐变骨架屏和线性进度条，禁止旋转大图标和复杂粒子特效。
- 弹窗弹出动画使用透明度 `0 -> 1`（0.2s）和轻微缩放 `scale(0.98) -> 1`，关闭时执行反向动画。
- 面板展开收起使用高度渐变过渡，时长 0.2s，不设置延迟。
- 批量任务长流程需要展示实时进度条和百分比文字，完成后弹出 Success Toast。
- 画布图层拖拽过程中，图层透明度降低至 70%，放置位置使用主色虚线框提示。

## 十、电商物料导出规范

### 10.1 图片格式

| 输出类型 | 默认格式 | 说明 |
| --- | --- | --- |
| 白底商品图 | JPG，质量 90 | 适合平台主图与详情页基础图 |
| 透明素材 | PNG | 保留透明通道 |
| 批量压缩 | WebP 可选 | 用于体积优化和批量投放 |

### 10.2 分辨率标准

- 线上电商展示图统一使用 72DPI。
- 印刷或高清商用素材可选 300DPI。

### 10.3 文件命名规则

命名格式：

```text
品类_产品名称_用途_宽×高
```

示例：

```text
美妆面霜_修护焕亮霜_主图 800×800
```

### 10.4 批量导出打包规则

批量导出时自动按类型归类为主图文件夹、场景图文件夹、详情页文件夹和营销海报文件夹。压缩包命名自动携带项目名称与导出时间。

## 十一、响应式适配规范

- 桌面端最小兼容宽度为 1280px，低于 1280px 时展示侧边折叠按钮。
- 当页面宽度不高于 1440px 时，侧边导航栏支持折叠，仅展示图标。
- 画布区域自适应压缩，最小宽度为 600px，宽度不足时底部出现横向滚动条。
- 弹窗在小宽度窗口中自动改为页面宽度的 90%，上下保留 32px 间距。

## 十二、企业版视觉约束

- 品牌色自定义仅管理员可编辑，普通操作员无权限。
- 自定义品牌色仅允许替换主交互色，AI 浅紫辅助色固定不可修改。
- 团队模板开启锁定后，LOGO 图层、品牌色和标准尺寸全部固化，删除 / 修改操作置灰禁用。
- 团队导出文件自动在底部增加轻量化品牌水印，水印透明度为 15%，颜色为 N4。

## 十三、前端组件落地规范

### 13.1 组件策略

- 不引入 Element Plus 作为视觉组件库，避免默认后台风格污染当前轻量创作工具视觉。
- 交互行为参考 Element Plus 的成熟组件模式，包括 focus、hover、active、disabled、loading、clearable、error、popover 外部点击关闭、Esc 关闭和键盘可达性。
- 视觉样式不得继承 Element Plus 默认主题，所有颜色、圆角、字号、阴影、间距必须来自本文档 token。
- 基础组件统一放在 `frontend/src/components/base`，业务页面优先使用基础组件或同名 `.ui-*` class，不再散落手写一次性按钮、Tab、搜索框样式。

### 13.2 基础组件清单

| 组件 | 责任 | 关键状态 |
| --- | --- | --- |
| `BaseButton` | 主按钮、次按钮、品牌辅助按钮、AI 按钮、文字按钮 | hover、active、focus-visible、disabled、loading |
| `BaseTabs` | 页面级 Tab 与内容分类 Tab | active、disabled、键盘左右切换 |
| `BaseSearchInput` | 顶部大搜索框和工具搜索框 | focus、clearable、loading、右侧上传 / 相机动作 |
| `BaseIconButton` | 图标按钮、工具栏按钮 | aria-label、hover、active、disabled |
| `BaseCard` | 通用卡片容器 | hoverable、selected、loading |
| `BaseTag` | 状态标签、筛选标签、AI 标记 | default、brand、accent、success、warning、error |
| `BaseDropdown` | 筛选下拉、操作菜单 | 外部点击关闭、Esc 关闭、选中态 |
| `BaseModal` | 弹窗与确认框 | 遮罩、Esc 关闭、底部操作区 |
| `BaseToast` | 顶部轻提示 | success、warning、error、info、2s 自动关闭 |
| `BaseProgress` | AI 生成进度和批量任务进度 | determinate、indeterminate |

### 13.3 图标规范

- 图标库统一使用 `lucide-vue-next`。
- 图标风格统一线性 2px 描边，尺寸仅使用 16px、20px、24px。
- 常规图标颜色使用 N5，主操作图标使用品牌蓝，AI 图标使用 AI 紫，禁用图标使用 N3。
- 禁止为常规 UI 手写临时 SVG；确需自定义图形时，应先确认 lucide 中没有等价图标。

### 13.4 截图参考组件风格

- Tab：页面级大字号、强字重、底部短线选中态，避免胶囊式后台 Tab。
- 搜索框：横向大搜索条，左侧浅色图标块，右侧保留相机 / 上传图标按钮。
- 按钮：白底细边框为常规按钮，黑底为主按钮，hover 仅轻微边框加深和背景变化。
- 筛选项：图标 + 文案 + 下拉箭头，弱边框或无底色，保持工具栏轻量。
- 所有组件动效统一 `0.2s ease`，不得使用复杂动效或大面积渐变。
